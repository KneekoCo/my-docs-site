How to Build a Simple Task Tracker in Notion

Use this guide to build a clean and functional task tracker in Notion to manage your Docusaurus portfolio work.

Step-by-Step Instructions​

Step 1: Open Notion and Create a New Page​

  1. Go to the Notion homepage: https://www.notion.so
  2. In the left sidebar, click the + New Page icon.


Step 2: Add a Database Table​

  1. From the new page, click Database at the bottom of the screen.


Heads Up!

The database pop-up allows you to pick a prebuilt table, generate one with AI, or start from scratch.

  1. Choose Empty Database.


Need Column Help?

See the Database Column Types table at the end of this guide for a breakdown of each option.

  1. Click the title and rename the database to something like Task Tracker.


You now have a blank task database ready for configuration.

Step 3: Set Up Your Table Structure​

Start customizing your table by renaming columns and adding new ones to suit your task tracking needs.

Suggested Table Structure​

Column NameTypePurpose
Task NameTitleClearly describe the task
StatusSelectTrack task state: Not Started, In Progress, Done
PriorityMulti-SelectFlag priority: High, Medium, Low
Due DateDateTarget completion date
NotesText/URLExtra details, links, or commands
Tip

To change a column’s type, click its name → Property Type → choose the correct option.

Customize Columns​

  1. Click the first column (Title) and rename it to Task Tracker.

  2. Click Add Property to add a new column.



  1. From the popup, select Select.
Need Column Help?

See the Database Column Types table at the end of this guide for a breakdown of each option.



  1. Rename the new column to Status.


  1. Add the options: Not Started, In Progress, Done.

  2. Click + to add another column.



  1. Choose Multi-select.
Need Column Help?

See the Database Column Types table at the end of this guide for a breakdown of each option.



  1. Rename the column to Priority.


  1. Add your priority tags.

  2. Click + to add a column, and choose Date.

Tip

You can configure date columns to include time if needed.



  1. Rename the column to Due Date.


  1. Add one more column by clicking +, and select Text.


  1. Rename this column to Notes.


Database Column Types​

Use this reference to understand each available Notion column type and how to apply it in your task tracker.

Column TypePurpose / Use Case
TitlePrimary identifier for the row (usually the task or item name).
TextFreeform text notes or descriptions.
NumberStores numeric values. Supports formatting (currency, percentage, etc.).
SelectChoose one option from a list (e.g., status).

Multi-selectChoose multiple tags or categories from a list (e.g., High, Low, Urgent).

DateAssign single or range-based dates (optionally with time).
PersonAssign to one or more Notion collaborators.
Files & MediaAttach or embed documents, images, videos, or links.
CheckboxToggle for binary tracking like complete/incomplete.
URLStore clickable web links.
EmailCapture email addresses as mailto links.
PhoneSave phone numbers (supports tel: link behavior).
FormulaWrite custom logic (like calculated deadlines or priorities).
RelationLink to another Notion database or item.
RollupAggregate or calculate values from related entries.
Created TimeAuto-filled timestamp for row creation.
Created ByAuto-filled with the user who created the row.
Last Edited TimeTimestamp for the most recent edit.
Last Edited ByTracks the user who last edited the row.

Add Your Tasks​

Once your columns are created, begin entering your tasks:

  • Initialize Docusaurus project
  • Fix dark/light mode CSS
  • Deploy to GitHub Pages
  • Add Notion tracker guide (this!)
Pro Tip

Use keyboard shortcuts in Notion to speed up entry. For example, Cmd + Enter (Mac) or Ctrl + Enter (Windows) saves a row.

Use It!​

  • Sort by due date or priority to stay on top of deadlines
  • Filter by status to see what's in progress
  • Switch Views: Toggle to Board view using /board for a kanban-style workflow
Optional Views

Create multiple views (Table, Board, Calendar) for flexibility in how you manage your work.

Why Use This?​

  • Organize your portfolio development
  • Stay focused and accountable
  • Showcase your project management skills in interviews and client demos

Pro Tip​

Link this Notion tracker in your portfolio or Docusaurus site footer to show off your workflow.
